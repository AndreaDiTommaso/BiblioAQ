prerequisiti: aver installato xampp
1) creare il datbase tramite il file crea_db.sql.
2) modificare il file database.php all'interno della cartella BiblioAQ/api/config con i propri dati di accesso al db.
3) copiare la cartella BiblioAQ/api in xampp/htdocs, e rinominare la cartella BiblioAQ_api.
4) dal control pannel xampp avviare apache e mysql.
5) lanciare il comando "ionic serve -l" dalla cartella BiblioAQ.
